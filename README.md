# Title: Identification of Pneumonia from Pediatric X-rays
## Team members

    Sai Karthik (sa1K), Makayla Phillips (THEDragon-007)
    
## Project decription

Goal: The goal of this project is to use computer vision techniques and neural networks like CNNs to identify the pneumonia from a scan of an X-ray. Once that goal is achieved stretch goals include: identifying the severity of the condition and identifying the location of the pneumonia. 

Dataset: https://www.kaggle.com/datasets/yusufmurtaza01/pediatric-chest-xray-pneumonia-balanced-dataset
